# 青海空间数据检查工具

基于 PySide6 的桌面应用程序，用于检查空间数据的完整性和一致性。

## 功能特性

- 支持批量检查多个文件夹
- 自动检查水系数据与图层数据的匹配关系
- 详细的检查规则：
  - 水系图层：河流代码17位、重复检查
  - 断面平面位置L.shp：河流代码/名称+编号23位+CS格式
  - 防治对象分布P.shp：河流代码/名称+代码唯一性
  - 隐患要素分布L.shp：河流代码/名称+编号28位
- Excel报告导出
- 实时进度显示

## 技术栈

- PySide6 (Qt6) - GUI框架
- geopandas - 空间数据处理
- pandas - 数据处理
- openpyxl - Excel导出

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行程序

```bash
python main.py
```

## 打包为exe

```bash
pip install pyinstaller
pyinstaller qh_dcpj.spec
```

打包后的文件在 `dist/青海空间数据检查工具/` 目录下。

## 使用说明

1. 选择要检查的文件夹（包含多个子文件夹）
2. 选择水系Shp文件
3. 点击"开始检查"
4. 查看汇总统计和详细结果
5. 导出Excel报告

## 检查规则详情

### 水系图层
- 河流代码必须是17位
- 河流代码不能重复

### 断面平面位置L.shp
- 河流代码必须是17位
- 河流代码、河流名称必须在水系中
- 河流代码对应的河流名称必须与水系一致
- 编号必须是23位
- 编号前17位必须与河流代码一致
- 编号后5位必须与断面名称后5位一致
- 断面名称格式：河流名称+CS+3位数字

### 防治对象分布P.shp
- 河流代码必须是17位
- 河流代码、河流名称必须在水系中
- 河流代码对应的河流名称必须与水系一致
- 代码字段不能重复

### 隐患要素分布L.shp
- 河流代码必须是17位
- 河流代码、河流名称必须在水系中
- 河流代码对应的河流名称必须与水系一致
- 编号必须是28位
- 编号开头6位必须是数字（政区代码）
- 编号7-23位必须与河流代码一致
- 编号字段不能重复
