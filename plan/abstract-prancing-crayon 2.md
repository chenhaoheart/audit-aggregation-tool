# 附表与空间数据对比检查功能设计方案

## 背景

在成果报表展示页面添加空间数据校验功能，实现附表与shp图层属性表的双向匹配检查。

## 用户确认的需求

1. **数据对应关系**：
   - 附表1（防治对象名录）↔ 防治对象分布P.shp
   - 附表2（跨沟道路桥涵）/ 附表3（沟滩占地）↔ 隐患要素分布L.shp

2. **检查内容**：记录双向匹配

3. **匹配关键字**：代码+名称双重匹配

4. **文件夹结构**：和现有空间数据检查页面一致（多个子文件夹，每个子文件夹包含shp文件）

## 字段映射关系

### 附表1 ↔ 防治对象分布P.shp

| 附表1字段 | shp字段 | 说明 |
|-----------|---------|------|
| 5.名称 | 名称 | 防治对象名称 |
| 6.代码 | 代码 | 防治对象代码 |
| 1.县（区、市、旗）名称 | - | 县名 |
| 3.乡镇名称 | - | 乡镇名 |

### 附表2/3 ↔ 隐患要素分布L.shp

| 附表2/3字段 | shp字段 | 说明 |
|-------------|---------|------|
| 名称 | 名称 | 隐患要素名称 |
| 编码/编号 | 编号 | 隐患要素编码 |

## 验证方式

1. 启动程序，切换到成果报表展示页面
2. 加载附表数据，确认数据正确展示
3. 展开空间数据校验区域
4. 选择空间数据文件夹和水系文件
5. 点击开始校验，检查校验结果是否正确
6. 点击"点击查看详情"，查看不匹配的记录详情
7. 测试导出校验报告功能
