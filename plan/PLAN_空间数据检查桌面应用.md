# 空间数据检查桌面应用项目计划

## 1. 项目概述

**项目名称**: 青海空间数据检查工具 (qh_dcpj)
**项目目录**: `D:\github\qh_dcpj_py`
**项目目标**: 将现有的空间数据检查脚本转换为桌面客户端应用

## 2. 技术栈选择

### 核心技术栈

| 组件 | 选择 | 理由 |
|------|------|------|
| **GUI框架** | PySide6 (Qt6) | 现代专业外观，与PyQt6 API兼容，免费商用 |
| **数据处理** | 复用现有脚本逻辑 | geopandas, pandas, openpyxl |
| **打包工具** | PyInstaller | 成熟稳定，支持Windows |
| **Python版本** | Python 3.10+ | 兼容性良好 |

### 技术栈优势
- PySide6 提供现代原生外观
- 可直接复用现有检查逻辑
- 支持Excel表格展示
- 支持文件拖拽
- 可打包为独立exe

## 4. 功能模块设计

### 4.1 主界面功能

| 功能 | 描述 |
|------|------|
| 文件夹选择 | 选择要检查的文件夹 |
| 水系文件选择 | 选择水系shp文件 |
| 开始检查 | 执行检查任务 |
| 进度显示 | 显示检查进度 |
| 结果展示 | 在表格中显示检查结果 |
| 导出Excel | 将结果导出为Excel |

### 4.2 检查功能(复用现有逻辑)

| 图层 | 检查规则 |
|------|---------|
| 水系图层 | 河流代码17位、重复检查 |
| 断面平面位置L.shp | 河流代码/名称+编号23位+CS格式 |
| 防治对象分布P.shp | 河流代码/名称+代码唯一性 |
| 隐患要素分布L.shp | 河流代码/名称+编号28位 |

### 4.3 结果展示

- 每个图层单独一个Sheet
- 汇总统计Sheet
- 总体统计Sheet
