# 成果报表展示页面UI设计方案

## 背景

在现有空间数据检查工具中添加成果报表展示功能。用户要求设计成**两个独立页面**：
1. **空间数据检查**（现有功能）
2. **成果报表展示**（新功能）

## 用户确认的需求

1. **文件加载方式**：选择文件夹，程序自动查找3个附表Excel文件
2. **表头展示**：保持多级表头结构，使用Qt的合并单元格功能
3. **功能分离**：与空间数据检查分成两个独立页面

---

## 关键实现细节

### 多级表头实现

使用 `QTableWidget` 的 `setSpan()` 方法合并表头单元格：

```python
# 设置两行表头
table.setColumnCount(20)
table.setRowCount(len(data) + 2)  # 数据行 + 2行表头

# 合并第一行的单元格
table.setSpan(0, 0, 1, 10)  # "基础信息" 合并10列
table.setSpan(0, 10, 1, 4)  # "风险隐患要素类别" 合并4列
table.setSpan(0, 14, 1, 5)  # "风险隐患影响类型" 合并5列
```

### 勾选符号处理

Excel中的 `þ` 符号转为可视化的 `✓`：

```python
def parse_checkbox(value):
    if pd.isna(value):
        return ''
    return '✓' if str(value).strip() == 'þ' else str(value)
```

### 一对多数据合并

附表1中同一防治对象可能对应多个跨沟道路桥涵，使用 `groupby` 合并：

```python
# 按防治对象名称分组，将多个桥涵名称合并
df.groupby('名称')['跨沟道路桥涵'].apply(lambda x: '、'.join(x.dropna()))
```
